#  Intelligent Translator Agent API

A high-performance translation agent powered by LLMs, built with **FastAPI** and **LangChain**. 
Designed for academic and professional scenarios, featuring **Context-Aware Translation** and **Adaptive System Prompts**.

##  Key Features
- **High Performance**: Built on FastAPI, supporting asynchronous concurrency.
- **Robust Engineering**: Full error handling and strict data validation using **Pydantic**.
- **Adaptive Persona**: System prompt automatically adjusts tone based on target language (e.g., Formal for Academic English).
- **Auto-Documentation**: Integrated Swagger UI for easy testing.

##  Tech Stack
- **Framework**: Python 3.10+, FastAPI
- **LLM Integration**: OpenAI SDK / DeepSeek API
- **Validation**: Pydantic V2
- **Environment**: Python-dotenv
